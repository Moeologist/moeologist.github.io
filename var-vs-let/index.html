var

var 关键字当前定义域中定义一个变量，当变量已经存在与外层作用域时，var 仅仅对其赋值，就像前面演示的一样。

let 以及 const

let 关键字就像我们在在其他语言中所做的工作一样。不同作用域的同名变量视为不同的变量。值得一提的是，在同一个作用域下 var 可以重复声明一个变量，而 let 不能。

const 和 let 相同但不接受重赋值操作。
